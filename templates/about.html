About This System

Prediction System for Store A

This is the prediction system for Store A, based on customer data, using a machine learning model trained with federated learning.

How It Works

The prediction model is the result of a federated learning process where multiple models were trained on different datasets. This application uses the best performing and latest model from that process to make predictions on new customer data for Store A.

Technologies Used

  • Federated Learning: Flower (flwr)
  • Machine Learning: TensorFlow, Keras
  • Backend: FastAPI, Flask
  • Frontend: HTML, CSS, JavaScript
